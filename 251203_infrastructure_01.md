- AWS 클라우드 컴퓨팅 개념

클라우드 컴퓨팅 : 인터넷으로 컴퓨터 자원을 빌려 쓰는 기술, 서버나 저장 공간, 프로그램을 필요할 때 빌려씀
클라우드 장점 5가지:
1. 비용 절감
2. 쉬운 확장
3. 빠른 속도
4. 보안
5. 장소에 상관없음

클라우드의 종류
1. 퍼블릭 클라우드: 여러 사람이 함께 공유하며 사용,
2. 프라이빗 클라우드: 나만, ex) 사내 네트워트 서버, = 온프레미스
43. 하이브리드 클라우드: 퍼블릭 + 프라이빗

aws 글로벌 서비스
aws 리전, 가용영역 엣지 로케이션으로 나눠짐
리전: 국가 단위, 서울 도쿄 뉴욕 등등 aws 센서가 모여있는 큰 지역
가용역역: 도시 단위, 리전 안에 있는 작은 데이터 센터, 한 리전에 보통 2~3개 위치
엣지 로케이션: 센터 구축이 어려운 곳에 설치, cloud Front가 데이터를 빠르게 전달하기 위해 사용하는 작은 저장소
